#### Install Dependencies

```shell
pnpm install
```

#### Start the development environment

```shell
// like ios
pnpm tauri ios init

// start
pnpm tauri ios dev
```

#### Build

```shell
pnpm tauri ios build
```

#### Lint Code

```shell
pnpm lint
```

#### TS type checking

```shell
pnpm type-check
```

#### Code Formatting

```shell
pnpm format
```

#### Running unit tests

```shell
pnpm test
```

#### Run unit tests and generate test reports

```shell
pnpm test:coverage
```
